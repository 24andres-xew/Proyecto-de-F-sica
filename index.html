Sistema de riego plantas con arduino

Nuestro producto

1Introducción

Para el desarrollo del sistema de riego de plantas utilizaremos herramientas sujetas a la programación además de un sistema eléctrico, para así crear un ambiente adecuado para las plantas teniendo en cuenta las características importantes para el desarrollo y crecimiento de cada una de ellas. El Sistema de riego se planeó para que tenga dos funciones de manejo, la función manual que consiste en dar click a un botón que se encargará de activar, la función automática cuenta con un sistema programado en arduino, y utilizando sensores de calor y de humedad y que al determinar cierta temperatura se activa inmediatamente regando así las plantas. Además de un sistema de luces para cada una de las plantas.

2Formulación del Proyecto

Identificación del Problema

Los sistemas de riego actualmente son una solución a los problemas de sequía en la agricultura. Para enfrentar las prolongadas sequías que se presentan como consecuencia de intensos veranos, se requiere encontrar soluciones eficientes que permitan garantizar la vida y productividad de los cultivos, por tanto se utilizan los sistemas de riego de forma eficiente y productiva para simular una producción real de cultivo agrícola.

Descripción del Proyecto

Con el proyecto de sistema de riego se quiere hacer una simulación real de cómo los factores físicos del agua, el aire, la temperatura y humedad afectan dentro de un sistema automático y como poder capturar esos datos en tiempo real, con el propósito de interactuar con las físicas hidráulicas y de temperatura; para presentar el proyecto con diferentes funcionalidades que hacen que el proyecto tome forma con los prototipos que se presentan. El proyecto busca cumplir con el objetivo de documentar el comportamiento cerrado de las físicas dentro de un entorno controlado, además de comenzar a entender poco a poco como funciona la naturaleza en su estado normal y como podemos manipularla dentro de la vida cotidiana y el trabajo.

Explicación de Materiales

Sensor DTH11

dispone de 3 pines, un pin para la entrada de tensión, otro pin para la masa, y otro pin para la transferencia de datos. Es capaz de transmitir la información hasta 20 metros de distancia, la tensión de alimentación es de 3 a 5.5 Voltios en continua, con una corriente de trabajo de 2.5 miliamperios.

Pantalla LCD

Una pantalla de cristal líquido o LCD (acrónimo del inglés liquid crystal display) es una pantalla delgada y plana formada por un número de píxeles en color o monocromos colocados delante de una fuente de luz reflectora.

Bomba de Agua

Esto es una bomba casera, está compuesta con lo siguiente: 2 tapas de tarro de café, la mitad de un lapicero, silicona, cinta negra, un motor de 10v y por dentro tiene 1 palillo de bombon, las aspas utilice tapas de lavar loza. Su creación fue la siguiente: -Le hizo huecos a las tapas de tarro, después pegó el motor a una tapa, y dentro pegó el palillo de bombón, le pegó las 4 aspas, después de eso pegó la mitad del lapicero y por último pegó la otra tapa y quedó como muestra en la foto.

Relé

(En inglés, relay) es un dispositivo que es utilizado para controlar el encendido y apagado de dispositivos que requieren altos voltajes o altas corrientes mediante una señal de corriente continua como las proporcionadas por los pines de Arduino UNO o ESP8266.

Botón Pulsante

Un pulsador eléctrico o botón pulsador es un componente eléctrico que permite o impide el paso de la corriente eléctrica cuando se aprieta o pulsa. El pulsador sólo se abre o se cierra cuando el usuario lo presiona y lo mantiene presionado. Al soltarlo vuelve a su posición inicial.

Arduino

Es utilizado como un microcontrolador, cuando tiene un programa descargado desde un ordenador y funciona de forma independiente de éste, y controla y alimenta determinados dispositivos y toma decisiones de acuerdo al programa descargado e interactúa con el mundo físico gracias a sensores y actuadores.
